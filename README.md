# Pressure & Thermal Gradients
Interactive Chart of Lithostatic Pressure and Geothermal Gradients along the Depth

## About the plots
Inspired by subaqueous–subaerial eruption dynamics described by Cas and Simmons (2018; https://doi.org/10.3389/feart.2018.00198), and by regional metamorphic thermal gradients as synthesised in Spear’s Metamorphic Phase Equilibria and P–T–t Paths (1993), together with relevant metamorphic frameworks discussed in Hoinkes et al. (2005, DOI: 10.1029/2005TC001796).
